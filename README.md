# Automobile SQL Portfolio
SQL portfolio for automobile sales, service and CRM analysis

This repository contains a SQL portfolio based on the automobile domain.

The project demonstrates how SQL can be used to solve real-world business problems related to:
- Vehicle sales
- Service and maintenance
- Customer relationship management (CRM)
- Call planning and dealer follow-ups

The focus of this portfolio is on writing clean, readable SQL queries and explaining the business logic behind them.
